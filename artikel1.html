output dan input c++

input dalam c++ merujuk pada data yang dimasukkan ke dalam program, seperti masukan dari pengguna atau sumber eksternal. output adalah hasil atau data yang dihasilkan oleh program dan ditampilkan kepada pengguna atau disimpan dalam berkas.


1. berikut ini contoh kode output:


#include

using namespace std;


int main() {

  cout << "halo dokter";

  return 0;

}



2. berikut ini contoh kode input:


#include

using namespace std;


int main() {

  string nama;

  cin >> nama;

  return 0;

}
