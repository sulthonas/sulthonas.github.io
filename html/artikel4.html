kalkulator sederhana c++

kalkulator c++ adalah program yang memungkinkan pengguna untuk melakukan operasi perhitungan matematika. dalam bahasa pemrograman c++, kalkulator biasanya menerima masukan angka dan operasi matematika, lalu menghitung dan menampilkan hasilnya. contoh sederhana kode untuk kalkulator penjumlahan, pengurangan, perkalian, dan pembagian telah diberikan dalam jawaban sebelumnya.


1. berikut ini contoh kode kalkulator sederhana:


#include

using namespace std;


int main() {

  double angka_pertama, angka_kedua;

  char operator;


  cout << "masukkan angka pertama: ";

  cin >> angka_pertama;


  cout << "masukkan angka kedua: ";

  cin >> angka_kedua;


  cout << "masukkan operator (+, -, *, /): ";

  cin >> operator;


  if(operator == '+') {

    cout << angka_pertama << " + " << angka_kedua << " = " << angka_pertama + angka_kedua;

  } else if(operator == '-') {

    cout << angka_pertama << " - " << angka_kedua << " = " << angka_pertama - angka_kedua;

  } else if(operator == '*') {

    cout << angka_pertama << " * " << angka_kedua << " = " << angka_pertama * angka_kedua;

  } else if(operator == '/') {

    cout << angka_pertama << " / " << angka_kedua << " = " << angka_pertama / angka_kedua;

  }

  return 0;

}
